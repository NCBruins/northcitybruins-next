## How to Use This Website

### Quick Start
1. Extract the zip file to your computer
2. Double-click `index.html` to open in your browser
3. Click through the navigation to see all pages

### Upload to Web Hosting
1. Extract all files
2. Upload everything to your web host via FTP or cPanel
3. Keep the folder structure intact:
   - `/css` - stylesheets
   - `/fonts` - web fonts
   - `/images` - all images and graphics
4. Your site will be live at your domain!

## Customization Guide

### Replace Images
The template uses placeholder sports images. Replace these with your own:
- **Logo**: `/images/logo.png` (your Bruins logo is already there!)
- **Hero image**: `/images/football.png` (replace with basketball action shot)
- **Program images**: `/images/ball.png`, `/images/shoes.png`, etc.
- Keep the same filenames or update the HTML references

### Update Text Content
All text can be edited in the HTML files:
1. Open any `.html` file in a text editor (Notepad++, VS Code, or even Notepad)
2. Find the text you want to change
3. Edit and save
4. Refresh your browser to see changes

### Modify Colors
Your team colors are already applied! But if you want to tweak:
1. Open `/css/style.css`
2. Search for:
   - `#003366` (navy blue)
   - `#FFD026` (gold)
3. Replace with your preferred colors

### Add Social Media Links
Update the footer social links in each HTML file:
```html
<a href="YOUR_FACEBOOK_URL" class="facebook">&nbsp;</a>
<a href="YOUR_TWITTER_URL" class="twitter">&nbsp;</a>
<a href="YOUR_INSTAGRAM_URL" class="googleplus">&nbsp;</a>
```
