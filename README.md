# North City Bruins — Sleek v2 (GitHub Pages)

A simplified, modern version of the Bruins site (less busy).

## Deploy
Put these files in your repo root and enable:
Settings → Pages → Deploy from a branch → main / (root)

## Update links + dates
Edit `script.js`:
- `registrationUrl`
- `volunteerUrl`
- `seasonRangeText`
- `dates.*`
