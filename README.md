# North City Bruins — Youth Basketball League (GitHub Pages)

A clean, modern static website template (HTML/CSS/JS) designed for GitHub Pages.

## Quick start (local preview)
1. Download this folder.
2. Open `index.html` in your browser, **or** run a tiny server:
   - Python:
     - `python -m http.server 5500`
     - Visit `http://localhost:5500`

## Deploy to GitHub Pages
1. Create a new repo (e.g. `north-city-hoops`).
2. Upload these files to the repo root.
3. In GitHub: **Settings → Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` (or `master`) / root
4. Your site will be available at:
   - `https://<your-username>.github.io/<repo-name>/`

## Customize
- Update program copy, schedule, and contact info in `index.html`.
- Change dates and form links in `script.js`:
  - `registrationUrl`
  - `volunteerUrl`
  - season date strings

## Forms (contact)
This template uses a demo form (no backend). Common options:
- Formspree
- Basin
- Getform
- Google Forms (link out)

## Assets
- `assets/logo.png` is your league logo.
