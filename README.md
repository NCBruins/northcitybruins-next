# North City Bruins — Sleek (GitHub Pages)

Modern, minimal site for North City Bruins with GA4 already installed.

## Deploy
Upload these files to your repo root and enable:
Settings → Pages → Deploy from a branch → main / (root)

## Update links + dates
Edit `script.js`:
- `registrationUrl`
- `volunteerUrl`
- `seasonRangeText`
- `dates.*`
