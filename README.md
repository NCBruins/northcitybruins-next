# North City Bruins Website

Welcome to the North City Bruins Toronto Youth Basketball Team website!

## Overview

This website has been customized with your team's branding, featuring:
- **Team Colors**: Navy Blue (#003366) and Gold (#FFD026) matching your logo
- **Your Logo**: Prominently displayed in the header
- **Basketball-themed Content**: All pages updated with relevant basketball program information

## Pages Included

1. **Home (index.html)**: Main landing page with team introduction and quick links
2. **About Us (about.html)**: Information about the team's mission and coaching philosophy
3. **Programs (product.html)**: Detailed list of all basketball programs offered
4. **News (blog.html)**: Latest updates and announcements
5. **Contact (contact.html)**: Contact form and location information

## How to Use

### Option 1: Open Locally
1. Extract the zip file to your computer
2. Open `index.html` in any web browser
3. Navigate through the site using the top menu

### Option 2: Upload to Web Hosting
1. Extract the zip file
2. Upload all files to your web hosting service via FTP or cPanel
3. Ensure all files maintain their folder structure:
   - `/css` - stylesheet files
   - `/fonts` - custom web fonts
   - `/images` - all images including your logo
   - `/js` - JavaScript files

## Customization Tips

### Updating Content
- All text content can be edited directly in the HTML files
- Open any `.html` file in a text editor (like Notepad++ or VS Code)
- Find the text you want to change and update it
- Save the file and refresh your browser to see changes

### Changing Images
- Replace image files in the `/images` folder with your own
- Keep the same filenames, or update the HTML to reference your new filenames
- Your logo is located at: `/images/logo.png`

### Modifying Colors
- Main color styles are in `/css/style.css`
- Current team colors:
  - Navy Blue: `#003366`
  - Gold: `#FFD026`
- Search and replace these color codes to change the color scheme

### Adding Social Media Links
- Update the social media links in the footer of each page
- Currently set to placeholder links - replace with your actual social media URLs

## Technical Notes

- The website is fully responsive and works on mobile devices
- Compatible with all modern browsers
- Uses custom web fonts for professional typography
- No backend/database required - pure HTML/CSS/JavaScript
